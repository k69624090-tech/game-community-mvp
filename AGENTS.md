# AGENTS.md

## プロジェクト概要
このプロジェクトは、初心者〜ライト層向けのゲーム感想コミュニティサイトのMVPです。
攻略中心ではなく「感想・共感・雑談」を重視した交流サイトを構築します。

対象ゲーム（初期）:
- ポケモン系
- Minecraft
- あつまれ どうぶつの森
- マリオカート ワールド
- スプラトゥーン3

## 最重要ルール
- 実装前に構成（フォルダ・DB・画面・実装ステップ）を提示
- MVP優先（最小構成で動くこと）
- 不要機能を追加しない
- 既存コードを壊さない
- シンプルで読みやすいコードを優先

## 技術スタック
- Next.js（App Router）
- TypeScript（strict）
- Tailwind CSS
- Supabase（Auth / DB）

## 必須機能（MVP）
- 認証（Supabase Auth）
- ゲーム一覧/詳細
- 投稿作成 / 一覧 / 詳細
- コメント
- リアクション
- ネタバレフラグ
- プロフィール
- 通報（簡易）
